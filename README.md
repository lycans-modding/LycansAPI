# LycansAPI

## À propos

LycansAPI est une collection de plusieurs librairies qui fonctionnent comme des plugins BepInEx.

Elles ne modifient pas le comportement du jeu mais proposent une manière unifiée de réaliser des tâches récurrentes dans la création de mods.

## Installation

Cette API s'installe sur Lycans comme n'importe quel autre mod, vous pouvez en apprendre plus en lisant le [Wiki Modding](https://lycans-modding.github.io/LMWiki/Jouer/Installer-des-mods/)
